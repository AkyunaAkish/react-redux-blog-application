### Setup

```
$ createdb react_blog (if you have postgreSQL on your machine)
- Setup .env file with connection string to your postgreSQL database
$ npm install -g knex webpack webpack-dev-server nodemon
$ npm install
$ webpack
$ cd dist && bower install
$ cd .. && nodemon
```
